# 백엔드 서버 실행 가이드

## 로컬에서 백엔드 실행하기

### 방법 1: uv 사용 (권장)
```bash
# 프로젝트 루트 디렉토리에서
uv run src/main.py
```

### 방법 2: Python 직접 실행
```bash
# 프로젝트 루트 디렉토리에서
cd src
python main.py
```

### 방법 3: uvicorn 직접 사용
```bash
# 프로젝트 루트 디렉토리에서
uvicorn src.main:app --host 0.0.0.0 --port 8000 --reload
```

## 백엔드 서버 확인

백엔드가 정상적으로 실행되면:
- 브라우저에서 `http://localhost:8000/` 접속
- `{"message":"Hello, World!"}` 응답 확인

## 프론트엔드 설정

로컬에서 백엔드를 실행하는 경우:
- 프론트엔드는 자동으로 `http://localhost:8000`을 사용합니다
- 별도 설정 불필요

원격 서버(3.35.8.64)를 사용하는 경우:
- 환경 변수 설정: `VITE_API_BASE_URL=http://3.35.8.64`
- 또는 `ssukssuk-closet/src/lib/api.ts` 파일 수정
